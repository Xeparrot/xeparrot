# WebAuthn Login Verify



## Endpoint
https://v3-shopper-api.adup.io/api/v1/web_authn/login

## Method
`Method: POST`

## Authorization

Type: Bearer Token

## Query Parameters

Parameter | Type | Required | Description
--------- | ------- |  ------- | -----------
challenge_data | text | Yes | You need to add challenge data
challenge_id | init | Yes | Challenge ID
